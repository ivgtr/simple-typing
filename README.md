# simple-typing

入力の経過は確認しないシンプルなタイピングゲームです。
音声入力とキーボード入力の速度をなるべく同条件で比較するのをモチベーションとしています。

## 概要

simple-typingは、タイピング速度を測定するためのミニマルなWebアプリケーションです。
従来のタイピングゲームと異なり、入力途中の正誤判定を行わず、最終的な入力結果のみを評価します。
これにより、音声入力とキーボード入力を公平に比較できる環境を提供します。

## 主な特徴

- 📝 **シンプルな入力体験**: 入力途中の正誤判定なし
- ⚡ **速度測定**: タイピング速度（WPM/CPM）を計測
- 🎤 **音声入力対応**: 音声入力とキーボード入力の両方に対応
- 📊 **結果表示**: 入力完了後に正確性と速度を表示
- 🏆 **スコア算出**: 正確性と速度を組み合わせた総合スコア
- 🎨 **モダンなUI**: Tailwind CSSによる洗練されたデザイン

## セットアップ

### 必要な環境

- Node.js 20.x 以上
- npm

### インストール

```bash
# リポジトリをクローン
git clone https://github.com/ivgtr/simple-typing.git
cd simple-typing

# 依存関係をインストール
npm install
```

### 開発

```bash
# 開発サーバーを起動
npm run dev
```

ブラウザで `http://localhost:5173` を開いてアプリケーションにアクセスできます。

### ビルド

```bash
# 本番用ビルド
npm run build

# GitHub Pages用ビルド
npm run build:gh
```

## スコア算出方法

入力完了後、以下の指標を基に総合スコアを算出します：

### 測定指標

1. **正確性（Accuracy）**
   - 正しく入力された文字数 ÷ 対象文章の総文字数 × 100
   - 0〜100%の範囲で表示

2. **速度指標**
   - **WPM (Words Per Minute)**: 1分あたりの入力単語数
     - 計算式: `(入力文字数 ÷ 5) ÷ 経過時間（分）`
     - 日本語では5文字を1単語として換算
   - **CPM (Characters Per Minute)**: 1分あたりの入力文字数
     - 計算式: `入力文字数 ÷ 経過時間（分）`

3. **総合スコア**
   - 正確性と速度を組み合わせて算出
   - 計算式: `スコア = (正確性 × 速度係数) + ボーナス`
   - 速度係数は入力方法（キーボード/音声）に応じて調整可能
   - 高精度かつ高速な入力ほど高スコアを獲得

### スコアの活用

- 過去の自分のスコアと比較して上達を確認
- 異なる入力方法（キーボード vs 音声）のパフォーマンス比較
- 練習による改善の可視化

## 使い方

1. アプリケーションを開く
2. 表示された文章を入力する（キーボードまたは音声入力）
3. 入力が完了したら結果を確認する
4. タイピング速度（WPM/CPM）、正確性、総合スコアが表示される
5. 「もう一度挑戦する」ボタンで再挑戦できる
