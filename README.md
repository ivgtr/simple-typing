# simple-typing

入力の経過は確認しないシンプルなタイピングゲームです。
音声入力とキーボード入力の速度をなるべく同条件で比較するのをモチベーションとしています。

## 概要

simple-typingは、タイピング速度を測定するためのミニマルなWebアプリケーションです。
従来のタイピングゲームと異なり、入力途中の正誤判定を行わず、最終的な入力結果のみを評価します。
これにより、音声入力とキーボード入力を公平に比較できる環境を提供します。

## 主な特徴

- 📝 **シンプルな入力体験**: 入力途中の正誤判定なし
- ⚡ **速度測定**: タイピング速度（WPM/CPM）を計測
- 🎤 **音声入力対応**: 音声入力とキーボード入力の両方に対応
- 📊 **結果表示**: 入力完了後に正確性と速度を表示
- 🎨 **モダンなUI**: Tailwind CSSによる洗練されたデザイン

## 技術スタック

- **フレームワーク**: [Svelte](https://svelte.dev/) - リアクティブなUIフレームワーク
- **ビルドツール**: [Vite](https://vitejs.dev/) - 高速な開発環境とビルド
- **スタイリング**: [Tailwind CSS](https://tailwindcss.com/) - ユーティリティファーストのCSS
- **デプロイ**: GitHub Pages - 自動デプロイ

## セットアップ

### 必要な環境

- Node.js 20.x 以上
- npm

### インストール

```bash
# リポジトリをクローン
git clone https://github.com/ivgtr/simple-typing.git
cd simple-typing

# 依存関係をインストール
npm install
```

### 開発

```bash
# 開発サーバーを起動
npm run dev
```

ブラウザで `http://localhost:5173` を開いてアプリケーションにアクセスできます。

### ビルド

```bash
# 本番用ビルド
npm run build

# GitHub Pages用ビルド
npm run build:gh
```

## デプロイ

このプロジェクトはGitHub Actionsを使用して自動的にGitHub Pagesにデプロイされます。
`main`ブランチにプッシュすると、自動的にビルドとデプロイが実行されます。

## 使い方

1. アプリケーションを開く
2. 表示された文章を入力する（キーボードまたは音声入力）
3. 入力が完了したら結果を確認する
4. タイピング速度と正確性が表示される
